Represents a scalar field in a model.

Name of the field.

The field’s type.

The field’s arity.

The database internal name.

§default_value: Option<DefaultValue>

The default value.

§documentation: Option<String>

Comments associated with this field.

§is_generated: bool

signals that this field was internally generated (only back relation fields as of now)

§is_updated_at: bool

If set, signals that this field is updated_at and will be updated to now() automatically.

§is_ignored: bool

Indicates if this field is ignored by the Client.
